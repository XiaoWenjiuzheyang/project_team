# 项目部学习文档

### 说明

* 此文档适合有HTML、CSS、JavaScript基础的同学观看。
* 此文档的目的是帮助同学们找到学习方向，而非介绍所有技术细节。
* 通过学习此文档的内容，可以达到独立完成一个内容发布系统或简单的电商系统。

### 学习步骤

1. 网页重构：还原PC端和手机端页面
2. JavaScript：变量、数据类型、控制流程语句、对象、函数、ES2015
3. jquery:常见的网页效果
4. git：常用命令，项目管理
5. vue:组件化模块化的开发
6. npm：常用命令
7. node:eggrestful接口，模板渲染
8. mysql:数据增删改查
9.  sequelize：egg对mysql实现增删改查
10. ajax:axios数据交互
11. 项目开发与部署
12. 真实项目
